# Tomshley Magicroot SBT

SBT plugins for Tomshley project conventions, settings, and templates.

This repository is part of the **Tomshley – OSS IP Division** and is maintained by **Tomshley LLC**.

---

## Overview

Magicroot SBT provides a set of opinionated SBT plugins that encode Tomshley project conventions:

- **magicroot-common** — Shared utilities and base definitions
- **magicroot-sbt-common** — Common SBT plugin infrastructure
- **magicroot-sbt-projectsettings** — Standardized project settings
- **magicroot-sbt-projecttemplate** — Project template scaffolding

---

## Usage

Add the plugin to your `project/plugins.sbt`:

```scala
addSbtPlugin("com.tomshley" % "magicroot-sbt-projectsettings" % "<version>")
```
